De opleiding

In het studiomodel werken studenten onder begeleiding van coaches en experts uit het vakgebied aan praktijkgerichte opdrachten.

De Hogeschool van Amsterdam start in het studiejaar 2021-2022 met de nieuwe opleiding Frontend Design & Development.

In de opleiding Frontend Design & Development leer je interactieve toepassingen maken voor het web. Met HTML, CSS en Javascript maak je echte opdrachten voor echte opdrachtgevers.

Alles wat je gaat doen doe je met code: Met code los je een parkeerprobleem op voor de Gemeente Amsterdam, met code ontwerp je voor de Openbare Bibliotheek een betere zoekfunctie, en met code help je de zorg te verbeteren samen met verpleegkundigen van het Academisch Medisch Centrum ...

Frontend designers en developers richten zich op web-design, visual interface design en frontend development:

  • Je leert over vormgeving, layout en typografie voor websites.
  • Je leert hoe je user-interface-principles kan toepassen voor een goede interactie.
  • Je codeert in HTML, CSS en Javascript, voor de browser en op de server.

HBO Associate Degree

In een Hackathon helpen frontend designers & developers studenten van andere opleidingen met het maken en testen van interactieve concepten. Bekijk een video van een Frontend Hackathon.

Een associate degree (Ad) is een tweejarige praktijkgerichte hbo-opleiding. Er zijn drie vormen van hbo onderwijs: hbo-associate degree, hbo-bachelor degree en hbo-master degree.

Een Ad is een goede keuze voor een student met een praktijkgerichte houding die een mbo 4-diploma heeft, graag wil doorleren en zich verder wil specialiseren of verbreden in een beroep.
